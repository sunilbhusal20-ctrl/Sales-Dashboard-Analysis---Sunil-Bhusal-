# 📊 Sales Dashboard Project
## 📌 Project Overview
This project is an interactive Sales Dashboard created using Microsoft Excel. The main objective of this project is to analyze and visualize employee-wise daily sales performance in a simple and effective way.

## 📁 Data Source
The dataset used in this project is collected from publicly available online platforms such as Kaggle and Wikipidia datasets. The data represents employee-wise daily product delivery/sales records.

## ⚙️ Data Preparation
- Raw data was collected and organized in Excel.
- The dataset includes:
  - Employee ID
  - Employee Name
  - Daily performance data (Sunday to Saturday)
- Data cleaning and structuring were done to make it suitable for analysis.

## 📊 Data Analysis
- Calculated weekly performance for each employee.
- Derived average daily sales for better comparison.
- Compared performance across different employees.
- Identified trends and variations in daily productivity.

## 📈 Dashboard Creation
- Built an interactive dashboard using Excel.
- Used different types of charts:
  - Pie Chart (for distribution analysis)
  - Bar/Column Charts (for comparison)
- Added interactive buttons and filters to improve usability.
- Designed the dashboard to provide clear and quick insights.

## 🎯 Key Features
- Employee-wise performance tracking
- Daily and weekly sales analysis
- Interactive and user-friendly dashboard
- Visual representation using charts

## 📌 Conclusion
This project demonstrates how raw data can be transformed into meaningful insights using Excel. It helps in understanding employee performance and supports better decision-making.
